# ✦ 粒子文字生成器

一个用纯 HTML + Canvas + JavaScript 构建的交互式粒子特效 Demo。

## 🎮 在线演示

👉 **[点击体验 Live Demo](https://你的用户名.github.io/my-demo)**

## ✨ 功能特点

- 输入任意文字，粒子自动排列成对应形状
- 鼠标悬停可扰动粒子，产生动态涟漪效果
- 实时显示粒子数量、帧率等统计信息
- 内置多个预设词句，一键切换

## 🛠 技术栈

- 纯原生 HTML / CSS / JavaScript
- Canvas 2D API 实现粒子系统
- 无任何第三方依赖，零配置运行

## 🚀 本地运行

直接双击 `index.html` 用浏览器打开即可，无需安装任何东西。
